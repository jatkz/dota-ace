    This article is about the store-bought Aghanim's Shard. For Tormentor's kill reward, see Aghanim's Shard - Consumable.

    Aghanim's Shard is an item purchasable at the Base Shop, under Consumables.

    Aghanim's Shard
    With origins known only to a single wizard, fragments of this impossible crystal are nearly as coveted as the renowned scepter itself.

    Additional Information

    [edit]
    • Cannot be purchased by heroes that already have the modifier_item_aghanims_shard modifier.
      • However, it can still be purchased via the hero's courier.

    Ability

    [edit]

    Ability Upgrade

    [edit]
    Ability Upgrade
    ▶️
    Ability
    Passive
    Affects
    Self
    Upgrades an existing ability or grants a new ability, unique to each hero.

    A hero can acquire the ability upgrade buff only once per match.
    Max Buff per Hero: 1
    Networth Gain: 1400
    • Details
    • Interactions
    • Misc.
    • Show All
    • modifier_item_aghanims_shard
      • Once the item enters the hero's inventory or backpack, it gets automatically consumed and converted into a permanent buff.
      • This only happens once. If the hero already has the buff, the item does not get consumed.
      • When consumed, the item is still counted towards the player's networth.
    • Aghanim's Shard status of allies can be seen in the scoreboard at all times.
      • Enemies' status can be seen as well, but it only updates once gaining vision over them.
    • Grants each hero one or multiple unique enhancements, ranging from ability upgrades, to hero changes, and sometimes even new abilities.
    Interactions: Self-Illusions
    Illusions copy the modifier on their creation. The upgrades fully work for them.
    Interactions: Ability Draft
    Most upgrades work normally for the player that drafted the abilities. For exceptions, see Ability Draft.
    Interactions: Spell Reflection
    Reflected abilities are upgraded based on the original caster's upgrades.
    Aghanim's Scepter & Shard UI element.
    • Has a dedicated interface element within the HUD, along with Aghanim's Scepter. The symbol lights up once Aghanim's Shard is acquired.
      • A tooltip pops up when pointing the mouse on this HUD element, showing the hero's ability upgrades.

    Upgraded Abilities

    [edit]
    Main Article: Aghanim's Shard/Upgraded Abilities
    • Aghanim's Shard first made its debut in Aghanim's Labyrinth, where they served as upgrades rewarded at the end of each room.
    • Aghanim's Shard is named after Agahnim, one of the main villains in The Legend of Zelda: A Link to the Past.
