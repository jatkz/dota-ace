    • This item is shown as a Team Item in the shop window when purchased by an ally.

    Soul Stealer[edit]

    Soul Stealer
    Ability
    Passive
    Affects
    Self
    Increases the item's bonus attack damage per debuff-affected enemy hero killed by Corruption.
    Max Stacks: 30
    Stack per Kill: 2
    Attack Damage Bonus per Stack: 1
    • Details
    • Interactions
    • Show All
      • The item grants 85 total flat bonus attack damage at max stacks.
      • Considers The Double, enemy illusions and creep-heroes as creeps.
        • The ability grant stacks to the main hero if desolator_buff is applied by the following units:
          • The Double.
          • Wukong's Command Soldiers.
          • Nothl Projection Shadow.
    • The stack gain has the following properties:
      • Only the first acquired Desolator grants stacks in the wielder's inventory or backpack.
      • Grant stacks to the most recent player who refreshed the desolator_buff.
      • The current stack is bound to the item.
    Interactions: Self-Illusions
    Non-functional.

    However, they copy the item's stack value upon creation for camouflaging purposes.

    Trivia[edit]

    • In DotA, this item is named Stygian Desolator, which is a reference to a weapon from Anarchy Online, the Stygian Desolator.
      • This name is now used by the neutral artifact Stygian Desolator.
