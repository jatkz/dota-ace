    Additional Information[edit]

    • The health regeneration bonus from multiple instances fully stacks.
    • Enchanted Mangoes dug up with Trusty Shovel are fully shareable, but cannot be sold.

    Ability[edit]

    Eat Mango[edit]

    Details
    • Has the following targeting functionalities:
        • Self-mana restores upon activation, even while the wielder is invulnerable.
        • Requires Ctrl to target an allied hero.
          • Considers clones, illusions, and creep-heroes as heroes.
          • Neither requires turning nor facing the targeted unit to cast, as long it is within the cast range.
          • Can be cast on untargetable units indirectly via portrait-casting.

    Trivia[edit]

    • Enchanted Mango's description is a reference to an old Weibo post by Icefrog, in which he mentioned that he loves mangoes.
    • It is believed that the item idea came from a Reddit thread where someone suggested a Tango that restores mana — and a user jokingly suggested that the item be named Mango.
    • Prior to the July 14, 2017 Patch, Enchanted Mango's active ability, Eat Mango, did not have an official name. The name was taken from the Dota 2 Wiki and officially implemented into Dota 2.
