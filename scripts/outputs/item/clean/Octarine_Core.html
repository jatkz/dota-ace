    Trivia[edit]

    • Octarine Core's name is an homage to the comic fantasy book series Discworld, paying tribute to the writer Terry Pratchett who died about a month prior to the item's release.
      • In the series, Octarine is a fictional color which is described as "resembling a flourescent greenish-yellow purple". The color can only be seen by wizards and by cats, which is also referenced by Octarine Core's item lore.
    • The components of Octarine Core before version 7.27 are the same as the components of the Pre-6.60Aghanim's Scepter.
