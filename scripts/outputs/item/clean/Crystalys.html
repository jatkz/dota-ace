    Proc Chance: 30%

    • Details

    • Show All
    • Critical strike procs have the following properties:
      • The proc is determined during the unit's attack point.
      • The proc chance of multiple sources stack.
        • However, when multiple critical strike sources proc simutaneously, the higher critical factor takes precedence.
      • Critical Strike uses pseudo-random distribution.
    • Increases attack damage by an average of 18%.
    Interactions: Self-Illusions
    Displays critical strike values before all damage manipulation sources for camouflaging purposes.
